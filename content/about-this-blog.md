## このブログについて

React ベースのフレームワークである Next.js については<br>
以前から興味がありました。

が、しかし、実際にどんなアプリを作るかとなると、<br>
なかなか思いつかないものです。

また、自分はこちらにあるような<br>
WordPress を使ったサイトを作ることが多かったのですが、

[Linux で Flutter 開発を試みる初心者の記録](https://flutter-newbie.com/)

CMS としては使いやすく、<br>
プラグインによるカスタマイズも容易なのですが、<br>
その反面、セキュリティ面でのリスクがあり、動作も重くなりがちです。

そこで、Next.js と Tailwind CSS による<br>
静的サイトジェネレーターを使って、<br>
Markdown で記事を書くことができるブログを作ってみることにしました。

## 使った技術

![test-image](/images/04_System76.png)

このサイトで使った技術は以下の通りです。

- Next.js 14
- Tailwind CSS
- Markdown のフロントマターを解析する gray-matter
- マークダウンを HTML に変換する remark
- アイコンとして、fontawesome

です。

ということでやってみました。
