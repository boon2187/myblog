## System76 の Linux をメインマシンとして使ってます

![test-image](/images/02_System76.png)

System76 というアメリカにある Linux マシンのメーカーの
Lemur Pro を購入してから１年が経ちました。

その前は、Dell の Latitude 5480(OS は Linux)と
M1 の MacBook Air を使っていました。

自分は、Adobe のソフトで
画像編集や動画編集をすることがないので、
まったく問題なく使えています。

というより、OS を意識することはほとんどありません。

むしろ、開発環境が整っているので、
会社で使っている Windows よりは使いやすいです。

## System76 ってどんなメーカー？

![test-image](/images/01_System76.png)
System76 とは、アメリカのコロラド州デンバーにある Linux がプレインストールされた PC の製造業者です。

また、Pop!\_OS という Ubuntu ベースの独自 OS をリリースしています。

ちなみに、Pop!\_OS は、
Linux のディストリビューションのランキングをつけているhttps://distrowatch.com/で、
常にトップ１０以内にランキングされている人気 Linux のひとつです。

また、Wired のレビューではかなりいい評価を頂いてます。

今回購入した Lemur Pro はこちらの Wired の記事にもあるように、なかなかの評価を得ています。

[Wired 誌のレビュー記事](https://wired.jp/article/system-76-lemur-pro-laptop/)

## メインマシンとして使ってみて

たまに使う Mac、会社で使う Windows と比べても、
不便に感じるところは、
自分の場合はまったくありません。

アプリケーションのインストールなんかは、
コマンドラインで簡単にできるので
むしろ、Mac や Windows よりも楽です。

Windows10 のサポートが切れるたときに、
Linux に移行するのもいいかもしれません、
と思う今日この頃です。

最後まで読んでいただきありがとうございます。
